## Core track work

The Core track established the foundational competencies expected of an entry-level security practitioner. Each piece below corresponds to a hands-on lab, exercise, or written analysis.

### Network and infrastructure security
- **Secure Network Design** — Designing segmented, defensible network architectures
- **Firewall Configuration on Kali** — Hands-on configuration and rule design
- **SSH Setup and Configuration** — Secure remote access fundamentals
- **Packet Sniffing** — Network traffic capture and analysis
- **Scanning Networks** — Reconnaissance and host discovery techniques

### Vulnerability and threat analysis
- **Nessus Scan** — Vulnerability scanning with a leading commercial scanner
- **Malicious Code** — Analysis of malware behavior and indicators
- **Tabletop Exercise** — Structured incident response planning

### Offensive security fundamentals
- **Password Cracking** — Password attack techniques and defenses

### Defensive practice
- **Endpoint Protection** — Host-based defense and hardening
- **Back-Up Lab** — Resilience and recovery practice

### Certification preparation
- **CompTIA Security+ Registration** — Documentation of certification pursuit (passed)
